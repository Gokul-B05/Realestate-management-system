# Real Estate Management System

A full-stack Real Estate Management System built using Spring Boot, React.js, and PostgreSQL.

## Live Demo

**Application:** https://realestate-frontend-iple.onrender.com

## Tech Stack

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Backend

* Java
* Spring Boot
* Spring Data JPA
* REST APIs

### Database

* PostgreSQL

## Features

* User Authentication
* Property Management
* Customer Management
* Property Search
* CRUD Operations
* Responsive User Interface
