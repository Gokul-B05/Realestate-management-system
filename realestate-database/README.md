# Real Estate Management System

A full-stack web application for managing properties, customers, and real estate transactions.

---

## Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Java
* Spring Boot
* Spring Data JPA
* REST API

### Database

* PostgreSQL

---

## Features

* User Authentication
* Property Management
* Customer Management
* Search and Filter Properties
* Dashboard
* CRUD Operations

## Installation

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm start
```

### 4. Database Setup

1. Create a PostgreSQL database.
2. Run `database/schema.sql`.
3. (Optional) Run `database/sample_data.sql`.
